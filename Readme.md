# Tetris (Pygame) 

一个遵循主流玩法规范的俄罗斯方块，使用 **Python + Pygame** 实现，**单文件代码**，开箱即玩。已内置 **中文字体自动匹配**（避免乱码）与 **NEXT 预览自适应布局**。

## ✨ 特性

- 标准 **7-袋随机器（7-bag）**
- **SRS 旋转**与**墙踢**（含 I 专用踢墙表）
- **影子（Ghost）**、**软降/硬降**、**Hold 暂存**
- **NEXT 预览**（默认 5 个，可一行改成 2 个）
- **锁定延迟（0.5s）**，地面移动/旋转可有限次重置（最多 15 次）
- **T-Spin 检测（含 Mini）**、**Combo**、**Back-to-Back** 加成
- **等级与重力**：每 10 行升 1 级，重力随等级加快
- **UI**：渐变背景、圆角方块、信息面板
- **本地最高分**：`tetris_highscore.json`

## 🕹️ 操作

- **移动**：`←/→`
- **软降**：`↓`（每格 +1 分）
- **硬降**：`Space`（每格 +2 分）
- **旋转**：`Z`（逆时针）、`↑/X`（顺时针）、`A`（180°）
- **暂存/交换**：`C`（每块只能用一次）
- **暂停/继续**：`P`　**重开**：`R`（或在结束后）　**退出**：`Esc`

## 🧮 计分（简述）

- 单/双/三/四消：100 / 300 / 500 / **800（Tetris）**
- T-Spin：Mini Single 200；Single **800**；Double **1200**；Triple **1600**
- **B2B**：连续 Tetris 或 T-Spin 给予 **1.5×** 加成  
- **Combo**：每次连续消行 +50×连击数  
- 软/硬降：每下落格 +1 / +2
